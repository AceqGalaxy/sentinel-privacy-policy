Sentinel — Privacy Policy

Effective date: October 23, 2025

1. Introduction

Sentinel is a Discord moderation and management bot designed to enhance the safety and organization of your server. This Privacy Policy explains how Sentinel handles data and respects user privacy.

2. Data Collection

Sentinel only collects the minimum amount of data required for functionality, such as user IDs, server IDs, and message content when necessary for moderation purposes. No sensitive or personal data is stored.

3. Data Usage

Data is used solely to provide moderation, logging, and administrative features within Discord servers. Sentinel does not sell, share, or use data outside of its intended purpose.

4. Data Storage and Security

Stored data is kept securely and deleted when no longer required. We use encryption and access controls to prevent unauthorized access.

5. User Rights

Server owners can request data deletion or removal of logs at any time by contacting the development team.

6. Contact

If you have questions or requests regarding this Privacy Policy, please contact us at: quantumopsdev@gmail.com
